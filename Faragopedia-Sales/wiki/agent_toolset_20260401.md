# agent_toolset_20260401

The built-in toolset identifier that enables all standard agent tools including Bash, file operations, and web search.

## Details

When enabled, provides the agent with: Bash (run shell commands), Read (read files), Write (write files), Edit (string replacement in files), Glob (file pattern matching), Grep (regex text search), Web Fetch (fetch URL content), and Web Search (search the web). Individual tools within the toolset can be disabled via the configs parameter.
