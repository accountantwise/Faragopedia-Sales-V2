# Standard Operating Procedure (SOP) Creation Use Case

A ChatGPT use case that involves converting Loom video transcripts into detailed SOPs.

## Details

Users can record a Loom video, then provide the transcript to ChatGPT with a prompt like: 'Here's a transcript from my video. Turn it into an SOP for new hires.' An advanced tip is to create a custom GPT that new employees use during onboarding for a more automated workflow.
