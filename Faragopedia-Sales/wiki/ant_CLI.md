# ant CLI

A dedicated command-line interface from Anthropic for creating and managing agents, environments, and sessions from the terminal.

## Details

Installable via Homebrew, Go, or direct curl download. Supports creating agents, environments, and sessions via command flags or YAML configuration passed via heredoc. Commands follow the pattern: ant beta:agents create, ant beta:environments create, ant beta:sessions create, ant beta:sessions:events send. Useful for developers already working in terminal environments like Claude Code.
