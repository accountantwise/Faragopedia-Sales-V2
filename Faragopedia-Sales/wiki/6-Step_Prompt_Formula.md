# 6-Step Prompt Formula

A structured framework for writing effective ChatGPT prompts, ordered from most to least important.

## Details

The six steps are: (1) Task - clearly define the end goal using action verbs like generate, write, or analyze; (2) Context - tailor responses by providing user background, success criteria, and setting; (3) Examples - provide examples to help ChatGPT mimic style, structure, and tone; (4) Persona - assign a specific character or expertise for ChatGPT to embody; (5) Format - specify output format such as bullet points, tables, markdown, or code blocks; (6) Tone - add emotional context such as casual, formal, witty, enthusiastic, or pessimistic.
